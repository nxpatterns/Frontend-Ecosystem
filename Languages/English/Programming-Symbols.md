# Programming Symbols

## Basic Symbols

```plaintext
(     = Open Parenthesis                        = Klammer auf (Offene Klammer)
)     = Close Parenthesis                       = Klammer zu (Geschlossene Klammer)
[     = Open Bracket                            = Eckige Klammer auf
]     = Close Bracket                           = Eckige Klammer zu
{     = Open Curly Bracket (Open Brace)         = Geschweifte Klammer auf
}     = Close Curly Bracket (Close Brace)       = Geschweifte Klammer zu
*     = Asterisk                                = Stern (Sternchen)
&     = Ampersand                               = Und-Zeichen (Et-Zeichen)
^     = Caret                                   = Dach (Zirkumflex)
%     = Percent Sign                            = Prozentzeichen
$     = Dollar Sign                             = Dollarzeichen
#     = Hash Sign (Number sign, Sharp, Pound)   = Rautenzeichen
@     = At Sign                                 = Klammeraffe
!     = Exclamation Mark (Bang)                 = Ausrufezeichen
~     = Tilde                                   = Tilde
'     = Single Quote                            = Einfaches Anführungszeichen
"     = Double Quote                            = Doppeltes Anführungszeichen
`     = Backtick (Back quote, Acute, Grave)     = Akzent
<     = Open Angle Bracket                      = Spitzklammer auf
>     = Close Angle Bracket                     = Spitzklammer zu
/     = Slash (Forward Slash)                   = Schrägstrich
\     = Back Slash (Backward Slash)             = Rückwärtsschrägstrich
-     = Dash (Line, Hyphen)                     = Bindestrich
_     = Underline (Underscore)                  = Unterstrich
.     = Dot (Point, Period)                     = Punkt
?     = Question Mark                           = Fragezeichen
|     = Pipe                                    = Senkrechter Strich
;     = Semi-Colon                              = Semikolon
,     = Coma                                    = Komma
:     = Colon                                   = Doppelpunkt
=     = Equal Sign                              = Gleichheitszeichen
+     = Plus Sign                               = Pluszeichen
-     = Minus Sign                              = Minuszeichen
*     = Asterisk                                = Sternchen
<>    = Less Than and Greater Than              = Kleiner als und größer als
<=    = Less Than or Equal To                   = Kleiner oder gleich
>=    = Greater Than or Equal To                = Größer oder gleich
!=    = Not Equal To                            = Ungleich
==    = Equal To                                = Gleich
===   = Strictly Equal To                       = Strikt gleich
!==   = Strictly Not Equal To                   = Strikt ungleich
||    = Logical OR                              = Logisches ODER
&&    = Logical AND                             = Logisches UND
++    = Increment                               = Inkrement
--    = Decrement                               = Dekrement
+=    = Add and Assign                          = Addieren und zuweisen
-=    = Subtract and Assign                     = Subtrahieren und zuweisen
*=    = Multiply and Assign                     = Multiplizieren und zuweisen
/=    = Divide and Assign                       = Teilen und zuweisen
%=    = Modulus and Assign                      = Modulo und zuweisen
```

### Context-related deviations

```plaintext
?     = Ternary Operator (Question Mark)       = Ternärer Operator (Fragezeichen)
:     = Ternary Operator (Colon)               = Ternärer Operator (Doppelpunkt)
->    = Arrow Operator                         = Pfeiloperator
::    = Scope Resolution Operator              = Bereichsauflösungsoperator
```
