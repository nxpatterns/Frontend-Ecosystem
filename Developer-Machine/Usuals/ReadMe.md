# Developer Machine

## Standard Programms (MacOS)

**CLI / Shell / Monitoring**

- `bat` – better `cat` with syntax highlighting
- `coreutils` – GNU basic CLI tools
- `eza`, `lsd` – modern `ls` replacements
- `tree` – directory trees
- `tmux` – terminal multiplexer
- `htop`, `iftop`, `mtr`, `glances`, `gotop` – system/network monitoring
- `midnight-commander` – terminal file manager
- `lolcat` – rainbow text (pure fun)

**Dev / Build / Languages**

- `node`, `pnpm` – JS runtime + package manager
- `biome` – JS/TS formatter + linter
- `cmake` – build system
- `python@3.11`, `pipx` – Python + isolated CLI installs
- `rust` – Rust toolchain
- `r`, `rbenv` – R language, Ruby version manager

**Cloud / Containers / Networking**

- `azure-cli` – Azure CLI
- `kubernetes-cli` (`kubectl`) – Kubernetes control
- `openvpn` – VPN client
- `openldap` – LDAP tools
- `nmap` – network scanner
- `telnet` – legacy TCP client
- `websocat` – WebSocket CLI

**Data / Databases**

- `postgresql@17` – PostgreSQL
- `pgcli` – Postgres CLI with autocomplete
- `jq` – JSON processor

**Media / Images / Fonts**

- `ffmpeg` – audio/video processing
- `imagemin stack`: `optipng`, `oxipng`, `pngquant`, `mozjpeg` – image compression
- `advancecomp` – recompress PNG/MNG
- `autotrace`, `potrace` – bitmap → vector
- `fontforge`, `fonttools`, `sfnt2woff`, `eot-utils` – font tooling
- `libass`, `frei0r`, `librist` – media/libs
- `pdf2svg`, `pdfcpu` – PDF tools
- `svgbob` – ASCII → SVG
- `qrencode` – QR codes

**Text / Docs / Diagrams**

- `pandoc` – document converter
- `graphviz`, `ditaa` – diagrams
- `highlight` – syntax highlighting
- `aha`, `ansifilter` – ANSI → HTML/text
- `ascii` – ASCII tables

**Security / Crypto**

- `mkcert` – local trusted TLS certs
- `oath-toolkit` – OTP / 2FA
- `exiftool` – metadata inspection

**Misc**

- `wget` – HTTP downloads
- `otree` – object tree viewer
- `espeak-ng` – text-to-speech
